🌿 Política de Privacidade — Doutor Café

Última atualização: 23 de junho de 2026
Resumo: O Doutor Café coleta apenas os dados necessários para o funcionamento do diagnóstico agrícola. Nunca vendemos seus dados. Você pode solicitar exclusão a qualquer momento.

1. Quem somos

O Doutor Café é um aplicativo de diagnóstico inteligente de doenças e pragas do cafeeiro, desenvolvido e operado por Weder Calixto Borges — MEI, CNPJ 67.223.948/0001-87, com sede em Cássia/MG, Brasil.

Contato: doutorcafe.app@gmail.com

2. Dados coletados

Coletamos apenas os dados necessários para o funcionamento do app:

  • Nome completo — para identificação do produtor
  • CPF — para controle de análises gratuitas e prevenção de fraudes
  • Celular — para recuperação de conta via PIN
  • E-mail — opcional, para comunicações sobre o serviço
  • Região geográfica — obtida via GPS do dispositivo, para adaptar o diagnóstico à região cafeeira do produtor
  • Fotos e vídeos de folhas/frutos — enviados pelo usuário para análise de IA. Não são armazenados permanentemente nos servidores
  • Histórico de diagnósticos — armazenado localmente no dispositivo e em nossos servidores para consulta futura

3. Como usamos os dados

  • Realizar o diagnóstico de doenças, pragas e deficiências nutricionais via inteligência artificial (Anthropic Claude)
  • Controlar o limite de análises gratuitas por CPF
  • Permitir recuperação de conta em novo dispositivo
  • Melhorar a precisão do serviço
  • Enviar comunicações sobre o serviço (apenas se você forneceu e-mail)

4. Compartilhamento de dados

Seus dados são compartilhados somente com:

  • Anthropic (anthropic.com) — as imagens enviadas são processadas pela IA Claude para gerar o diagnóstico. A Anthropic não armazena as imagens após o processamento
  • Mercado Pago — para processamento de pagamentos (somente dados de pagamento)
  • Railway / Vercel — infraestrutura de hospedagem dos servidores

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais.

5. Armazenamento e segurança

  • Os dados são armazenados em servidores seguros na nuvem (Railway/PostgreSQL)
  • As fotos enviadas para diagnóstico não são armazenadas permanentemente
  • O histórico de análises é armazenado localmente no seu dispositivo e em nossos servidores
  • Utilizamos conexões HTTPS para todas as comunicações

6. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acessar os dados que temos sobre você
  • Corrigir dados incorretos
  • Solicitar exclusão dos seus dados
  • Revogar o consentimento a qualquer momento
  • Portabilidade dos dados

Para exercer esses direitos, entre em contato: doutorcafe.app@gmail.com

7. Retenção de dados

Mantemos seus dados enquanto sua conta estiver ativa. Após solicitação de exclusão, os dados são removidos em até 30 dias.

8. Uso por menores

O Doutor Café é destinado a produtores rurais adultos. Não coletamos intencionalmente dados de menores de 18 anos.

9. Localização (GPS)

Solicitamos permissão de localização apenas para identificar a região cafeeira e adaptar o diagnóstico (ex: Mogiana, Sul de Minas, Cerrado Mineiro). O GPS não é usado para rastreamento contínuo.

10. Câmera e galeria

Solicitamos acesso à câmera e galeria apenas quando o usuário inicia uma análise. As imagens são enviadas para processamento e não ficam armazenadas permanentemente nos nossos servidores.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. Notificaremos usuários sobre mudanças significativas pelo e-mail cadastrado ou via aviso no aplicativo.
